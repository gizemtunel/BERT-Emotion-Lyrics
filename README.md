# Şarkı Sözlerinden Duygu Analizi (Sentiment Analysis)

Bu proje, şarkı sözlerini analiz ederek içerdikleri duyguları  sınıflandırmak amacıyla geliştirilmiştir. Doğal Dil İşleme (NLP) teknikleri kullanılarak hazırlanan model, metin tabanlı veriler üzerinde yüksek doğrulukla çalışmaktadır.
